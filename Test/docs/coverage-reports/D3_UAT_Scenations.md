Scenario 1: การค้นหาและบันทึกหอพักที่สนใจ (สำหรับนิสิต)
วัตถุประสงค์: เพื่อทดสอบว่าผู้ใช้งานทั่วไปสามารถค้นหาหอพักตามเงื่อนไขและบันทึกไว้ดูภายหลังได้

ขั้นตอนการทดสอบ:

เข้าสู่ระบบ (Login) ในฐานะผู้เช่า/นิสิต

ไปที่หน้า "ค้นหาหอพัก"

พิมพ์ชื่อทำเลหรือชื่อหอพัก (เช่น "บางแสน" หรือ "สุขสบาย")

เลือกตัวกรอง (Filter): ราคาไม่เกิน 5,000 บาท และ "มีเครื่องปรับอากาศ"

กดปุ่ม "ค้นหา"

เมื่อพบหอพักที่ถูกใจ ให้กดไอคอน "หัวใจ" เพื่อบันทึกเป็นรายการโปรด

ไปที่เมนู "รายการโปรด" เพื่อตรวจสอบว่าหอพักถูกบันทึกจริงหรือไม่

ผลที่คาดหวัง:
✓ ระบบแสดงผลเฉพาะหอพักที่ตรงตามเงื่อนไข (ราคาและแอร์)
✓ สามารถบันทึกและแสดงผลในรายการโปรดได้อย่างถูกต้อง

[ ] ผ่านการทดสอบและยอมรับระบบทั้งหมด
[ ] ผ่านการทดสอบบางส่วน (มีข้อแก้ไขตามหมายเหตุด้านบน)
[ ] ไม่ผ่านการทดสอบ

Scenario 2: การลงประกาศและจัดการสถานะห้องพัก (สำหรับเจ้าของหอ)
วัตถุประสงค์: เพื่อทดสอบว่าเจ้าของหอพักสามารถลงข้อมูลและควบคุมการแสดงผลของหอพักตัวเองได้

ขั้นตอนการทดสอบ:

เข้าสู่ระบบในฐานะ "เจ้าของหอพัก" (Owner)

ไปที่หน้า "จัดการหอพักของฉัน"

กดปุ่ม "เพิ่มหอพักใหม่" และกรอกรายละเอียด (ชื่อ, ราคา, สิ่งอำนวยความสะดวก, อัปโหลดรูปภาพ)

กด "บันทึก" และตรวจสอบว่าหอพักปรากฏบนหน้าเว็บหลัก

ทดลองเปลี่ยนสถานะห้องพักจาก "ว่าง" เป็น "ห้องเต็ม"

กลับไปดูที่หน้าเว็บหลักเพื่อเช็กว่าสถานะอัปเดตหรือไม่

ผลที่คาดหวัง:
✓ ข้อมูลและรูปภาพแสดงผลถูกต้องตามที่กรอก
✓ สถานะ "ห้องเต็ม" แสดงผลชัดเจนเพื่อป้องกันคนติดต่อซ้ำ

[ ] ผ่านการทดสอบและยอมรับระบบทั้งหมด
[ ] ผ่านการทดสอบบางส่วน (มีข้อแก้ไขตามหมายเหตุด้านบน)
[ ] ไม่ผ่านการทดสอบ

Scenario 3: การนัดหมายเข้าดูห้องพัก (End-to-End Workflow)
วัตถุประสงค์: เพื่อทดสอบระบบสื่อสารระหว่างผู้เช่าและเจ้าของหอพัก

ขั้นตอนการทดสอบ:

ผู้เช่ากดเข้าไปดูรายละเอียดหอพักที่สนใจ

กดปุ่ม "นัดดูห้องพัก" หรือ "สอบถามเพิ่มเติม"

กรอกข้อความ วันที่ต้องการนัดหมาย และเบอร์โทรศัพท์ แล้วกด "ส่ง"

สลับไป Login ในฝั่งเจ้าของหอพัก

เข้าไปที่เมนู "ข้อความตอบกลับ" หรือ "การนัดหมาย" เพื่อดูข้อมูลที่ส่งมา

ผลที่คาดหวัง:
✓ ระบบส่งข้อมูลนัดหมายไปยังเจ้าของหอพักได้ครบถ้วน
✓ เจ้าของหอพักสามารถเห็นข้อมูลติดต่อของผู้เช่าเพื่อโทรกลับได้

[ ] ผ่านการทดสอบบางส่วน (มีข้อแก้ไขตามหมายเหตุด้านบน)
[ ] ไม่ผ่านการทดสอบ
